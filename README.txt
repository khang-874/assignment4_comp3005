Setup instruction:
1. Create the university database in postgres or change the database in connection url in the application.properties file to your database at the spring.datasource.url
2. Change spring.datasource.username to your postgres username
3. Change spring.datasource.password to your postgres password
4. Run the script.sql to create table and populate the data
5. Run the Assignment4Application class
6. If fail, install maven and try again.
7. Go to your browser at localhost:8080/students/list to interact
